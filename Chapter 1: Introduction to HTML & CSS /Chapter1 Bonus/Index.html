HTML and CSS Resource Bonus

This guide provides a quick reference to essential HTML tags and CSS properties for web development.

HTML Tags

Below is a list of commonly used HTML tags, their purposes, and examples.

h1 to h6 - Headings

Headings are used to define the structure and hierarchy of content. h1 is the most important, h6 the least.

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Sub-subheading</h3>

p - Paragraph

Defines a paragraph of text.

<p>This is a paragraph of text.</p>

a - Anchor/Link

Creates a hyperlink to another page or resource.

<a href="https://example.com">Visit Example</a>

img - Image

Embeds an image in the document.

<img src="image.jpg" alt="Description of image">

ul and li - Unordered List

Creates a bulleted list.

<ul>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ul>

ol and li - Ordered List

Creates a numbered list.

<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>

div - Division

A generic container for grouping content.

<div class="container">
    <p>Content inside a div</p>
</div>

span - Inline Container

An inline container for styling or scripting.

<p>This is a <span style="color: red;">red span</span> inside text.</p>

strong - Strong Emphasis

Makes text bold to indicate importance.

<p>This is <strong>important text</strong>.</p>

em - Emphasis

Makes text italic to indicate emphasis.

<p>This is <em>emphasized text</em>.</p>

br - Line Break

Inserts a line break.

<p>First line<br>Second line</p>

hr - Horizontal Rule

Creates a horizontal line to separate content.

<p>Content above</p>
<hr>
<p>Content below</p>

form - Form

Creates a form for user input.

<form action="/submit" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <button type="submit">Submit</button>
</form>

input - Input Field

Creates an input field for user data.

<input type="text" placeholder="Enter text">
<input type="email" placeholder="email@example.com">
<input type="password" placeholder="Password">

button - Button

Creates a clickable button.

<button type="button" onclick="alert('Hello!')">Click Me</button>
<button type="submit">Submit Form</button>

table, thead, tbody, tr, th, td - Table Elements

Creates structured tables for data display.

<table>
    <thead>
        <tr>
            <th>Name</th>
            <th>Age</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>John</td>
            <td>25</td>
        </tr>
    </tbody>
</table>

header, footer, nav, main, section - Semantic Elements

HTML5 semantic elements for better structure and accessibility.

<header>
    <nav>Navigation menu</nav>
</header>
<main>
    <section>Content section</section>
</main>
<footer>Footer content</footer>

CSS Properties

Essential CSS properties for styling web pages.

color

Sets the color of text.

p {
    color: blue;
}

background-color

Sets the background color of an element.

body {
    background-color: lightgray;
}

font-family

Specifies the font family for text.

body {
    font-family: Arial, sans-serif;
}

font-size

Sets the size of the font.

h1 {
    font-size: 2em;
}

margin

Sets the margin around an element.

p {
    margin: 10px;
}

padding

Sets the padding inside an element.

div {
    padding: 20px;
}

border

Sets the border around an element.

img {
    border: 1px solid black;
}

text-align

Aligns text within an element.

h2 {
    text-align: center;
}

width and height

Sets the width and height of an element.

img {
    width: 300px;
    height: 200px;
}

display

Specifies the display behavior of an element.

.inline-block {
    display: inline-block;
}

flexbox - Flexbox Layout

Modern layout system for arranging elements.

.container {
    display: flex;
    justify-content: space-between;
    align-items: center;
    gap: 20px;
}

grid - Grid Layout

Two-dimensional layout system for complex designs.

.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 20px;
}

transform - Transform Property

Applies 2D or 3D transformations to elements.

.box {
    transform: rotate(45deg) scale(1.2);
    transition: transform 0.3s ease;
}

transition - Transition Property

Creates smooth animations for property changes.

.button {
    background: blue;
    transition: all 0.3s ease;
}
.button:hover {
    background: red;
}

box-shadow - Box Shadow

Adds shadow effects around elements.

.card {
    box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1);
    border-radius: 8px;
}

position - Position Property

Controls the positioning method of elements.

.absolute {
    position: absolute;
    top: 10px;
    right: 10px;
}
.relative {
    position: relative;
    top: 5px;
}

z-index - Z-Index Property

Controls the stacking order of positioned elements.

.overlay {
    position: absolute;
    z-index: 1000;
    background: rgba(0, 0, 0, 0.5);
}

opacity - Opacity Property

Sets the transparency level of an element.

.transparent {
    opacity: 0.7;
}
.fade-in {
    animation: fadeIn 1s ease-in;
}

End of Guide

For more information, refer to official documentation.
